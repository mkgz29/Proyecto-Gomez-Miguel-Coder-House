# Moto Travel - Proyecto Web (Full Stack) -(CH - RC)

Este proyecto es una "pagina web estetica" creada como parte de mi aprendizaje en la carrera de **Desarrollo Web Full Stack**.  
La temática elegida es **Moto Travel** 🏍️, un sitio orientado a mostrar modelos KTM, secciones informativas y simulación de tienda/contacto.

---

 Características principales
- 🌐 **Landing page con video de fondo** para generar impacto visual.  
- 🧭 **Barra de navegación** con secciones: Inicio, Nosotros, Tienda, Contacto y Envíos.  
- 🎨 **Estilos personalizados con CSS/Sass**, aplicando tipografías y efectos.  
- 📱 **Diseño responsivo**, adaptable a dispositivos móviles y escritorio.  
- 🔎 Barra de búsqueda incluida en el header.  

---

## 🛠️ Tecnologías utilizadas
- **HTML5** → estructura del sitio.  
- **CSS3 / Sass** → estilos y personalización.  
- **Bootstrap 5** → grillas y componentes responsivos.  
- **Git & GitHub** → control de versiones.
